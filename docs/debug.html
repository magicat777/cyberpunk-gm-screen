Option 2: Create a Pull Request

For a permanent fix, create a pull request with these settings:

base: master ← compare: fix/desktop-ui-navigation

Steps:

  1. Go to GitHub Pull Request page
  2. Set base to "master"
  3. Set compare to "fix/desktop-ui-navigation"
  4. Click "Create pull request"
  5. Add title: "Fix desktop UI interaction and navigation"
  6. Add description: "Fixes UI issues including panel drag-and-drop, dropdown menus, and navigation components"
  7. Click "Create pull request" again
  8. Click "Merge pull request"

How to Use the Bookmarklet

  1. Drag the "CP UI Fix" button to your bookmarks bar
  2. Visit Cyberpunk GM Screen
  3. Click the bookmarklet from your bookmarks bar
  4. The UI should now be fully functional
