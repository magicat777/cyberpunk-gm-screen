Keyboard Shortcuts

The Cyberpunk RED GM Screen supports a variety of keyboard shortcuts to enhance your productivity. Learning these shortcuts can significantly speed up your workflow during game sessions.

Global Shortcuts

These shortcuts work anywhere in the application:

Action Shortcut Description
Open Help F1 or ? Opens this help page
New Notes Panel Ctrl + N Creates a new notes panel
Save Layout Ctrl + S Saves the current panel layout
Auto-Organize Ctrl + O Automatically arranges panels
Fit to Window Ctrl + F Resizes and positions panels to fit the current window
Toggle Theme Ctrl + T Cycles through available themes
Font Settings Ctrl + , Opens the font settings panel
Focus Search Ctrl + / Focuses the search input if available

Panel Management

These shortcuts work with panels in the main interface:

Action Shortcut Description
Close Panel Esc Closes the currently focused panel
Next Panel Ctrl + Tab Focuses the next panel
Previous Panel Ctrl + Shift + Tab Focuses the previous panel
Move Panel Alt + Arrow Keys Moves the focused panel (requires keyboard focus on panel header)
Resize Panel Alt + Shift + Arrow Keys Resizes the focused panel
Maximize Panel Alt + M Toggles maximizing the focused panel

Notes Panel Shortcuts

These shortcuts work in the Notes panel text editor:

Action Shortcut Description
Bold Text Ctrl + B Toggles bold formatting for selected text
Italic Text Ctrl + I Toggles italic formatting for selected text
Underline Text Ctrl + U Toggles underline formatting for selected text
Heading 1 Ctrl + 1 Formats selected text as a level 1 heading
Heading 2 Ctrl + 2 Formats selected text as a level 2 heading
Bulleted List Ctrl + Shift + 8 Creates or formats as a bulleted list
Numbered List Ctrl + Shift + 7 Creates or formats as a numbered list
Save Notes Ctrl + S Manually saves the notes content (also saved automatically)

Dice Roller Shortcuts

These shortcuts work in the Dice Roller panel:

Action Shortcut Description
Roll Dice Enter Executes the current dice roll
Clear Results Ctrl + Del Clears the dice roll history
Repeat Last Roll Ctrl + R Repeats the last dice roll
Quick d6 Alt + 6 Quickly rolls a single d6
Quick d10 Alt + 0 Quickly rolls a single d10
