Privacy Policy

Data Collection and Storage

Client-Side Storage

The Cyberpunk RED GM Screen is a browser-based application that operates entirely on your device. We use the following local storage mechanisms:

  • Local Storage: Used to save your screen layout, notes, and preferences.
  • Session Storage: Used for temporary data during your session.
  • Cookies: Minimal cookies used only for essential functionality.

All data is stored exclusively on your device and is not transmitted to our servers.

Data We Do Not Collect

We do not collect any personal information, including:

  • Personal identification information
  • Contact details
  • Browser history
  • IP addresses
  • The content of your notes or game data

Third-Party Services

GitHub Pages Hosting

This application is hosted on GitHub Pages, which may collect basic usage analytics such as page views and referral sources. This data is anonymized and not linked to individual users.

For more information, please refer to GitHub's Privacy Statement.

External Links

This application may contain links to external websites or resources. We are not responsible for the privacy practices or content of these external sites.

Data Security

Since all your data is stored locally on your device, the security of this data depends on your browser's security measures and your device's integrity.

We recommend the following practices to protect your data:

  • Keep your browser updated to the latest version
  • Use security features on your device (password protection, encryption)
  • Clear browser data when using shared or public computers
  • Export and backup important game data regularly

Updates to This Policy

We may update this privacy policy from time to time. Any changes will be posted on this page with an updated revision date.
