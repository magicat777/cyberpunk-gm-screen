3. Responsive Design Test

4. Motion Preference Support

Animations and transitions now respect the user's prefers-reduced-motion setting.

You can toggle animations using the button below, or through your operating system's reduced motion setting.

Elements with animations/transitions:

  • Dropdown menus (slide animation)
  • Buttons (hover effect)
  • Panels (dragging and resizing)
  • Notifications (fade in/out)

5. Keyboard-accessible Panel Controls

Panel resize handles can now be used with keyboard (demonstration only - not fully functional)

Tab to the resize handle, press Enter to activate resize mode, then use arrow keys to resize. Press Escape when done.
