Help Guide

Getting Started

The Cyberpunk RED GM Screen is designed to be intuitive and easy to use. Here's how to get started:

Adding Panels

  1. Click on the Add Panel dropdown in the top toolbar
  2. Select the type of panel you want to add from the menu
  3. The panel will appear in the main content area
  4. Drag panels by their header to reposition them
  5. Resize panels using the resize handle in the bottom-right corner

Managing Layouts

You can save and load different panel layouts:

  1. Click on the Layout dropdown in the top toolbar
  2. Select Export Layout to save your current arrangement
  3. Use Import Layout to load a previously saved layout
  4. Toggle Auto-Save to automatically save your layout

Panel Types

Notes Panel

The Notes panel provides a rich text editor for keeping game notes:

  • Use the formatting toolbar to apply styling (bold, italic, lists, etc.)
  • Content is automatically saved to your browser's local storage
  • You can create multiple note panels for different topics

Dice Roller

The Dice Roller panel allows you to roll virtual dice:

  • Select the dice type (d6, d10, etc.) from the dropdown
  • Enter the number of dice to roll
  • View results with individual dice values and totals
  • Roll history is saved for the current session

Initiative Tracker

Keep track of combat turns with the Initiative Tracker:

  • Add characters with their initiative values
  • Sort automatically by initiative order
  • Mark current turn and track round number
  • Add notes or status effects to characters

Reference Panels

Access game rules and tables quickly with reference panels:

  • Rules Reference: Quick access to core game mechanics
  • Weapons Table: Stats for various weapons
  • Armor Table: Protection values and special properties
  • Critical Injuries: Effects and recovery times
  • Netrunning: Architecture and program references

Generator Panels

Generate content on the fly for your game:

  • NPC Generator: Create random characters with stats
  • Location Generator: Generate urban locations with descriptions
  • Random Encounter: Create unexpected situations

Night City Map

Interactive map of Night City with district information:

  • Click on districts to view information
  • Toggle points of interest
  • Add custom markers for campaign locations

Settings & Customization

Theme Switching

Change the visual appearance of the GM Screen:

  • Neon Synthwave: Vibrant neon colors with purple accents
  • Tech Noir: Green-on-black terminal style interface
  • Minimal: Cleaner, less distracting interface

Font Settings

Customize text appearance for better readability:

  • Adjust font size (Tiny, Small, Medium, Large)
  • Select font family (Monospace, Sans-serif, Display, Cyberpunk)
  • Settings are saved between sessions

Frequently Asked Questions

Q: Are my notes and layouts saved permanently?

A: Notes and layouts are saved in your browser's local storage. They will persist between sessions but can be lost if you clear your browser data. For long-term storage, use the Export Layout feature to save your layouts as files.

Q: Can I use this offline?

A: Yes, once loaded, the application can function offline. However, some features like external links will not be available without an internet connection.

Q: How do I report bugs or request features?

A: Visit the Feedback page to report issues or suggest improvements.

Q: Will my custom layouts work in future versions?

A: We strive to maintain compatibility, but major updates may require layout adjustments. Always keep backups of important layouts.

Q: Can I share layouts with other GMs?

A: Yes, exported layout files can be shared with other users who can then import them into their GM Screen.

Troubleshooting

Layout Problems

  • Panels not saving: Check if your browser has storage restrictions or is in private/incognito mode
  • Panel positioning issues: Try the "Fit to Window" option in the Layout menu
  • Missing panels: Use "Auto-Organize" to bring off-screen panels back into view

Performance Issues

  • Slow response: Reducing the number of open panels can improve performance
  • Browser freezing: Try clearing the application cache or using a different browser
  • Display glitches: Refresh the page or switch themes to reset visual elements

Reset Application

If you encounter persistent issues, you can reset the application:

  1. Open the Settings menu
  2. Select "Test Storage"
  3. Use the "Clear All Storage" option (Warning: This will delete all saved data)
