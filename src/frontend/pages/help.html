Help & Documentation

Getting Started

The Cyberpunk RED GM Screen is designed to make running your tabletop RPG sessions easier. Here's how to get started:

Basic Navigation

  • Panel System: The GM screen consists of resizable, draggable panels containing different tools and references.
  • Toolbar: Use the top toolbar to access global actions and settings.
  • Panel Controls: Each panel has controls in its header for minimize, maximize, close, and resize.

Customizing Your Layout

  • Drag panels by their headers to reposition them on screen.
  • Resize panels by dragging their corners or edges.
  • Save layouts using the Layout menu in the toolbar.
  • Reset to default if you need to start fresh.

Main Features

Rules Reference

Quick access to commonly referenced rules and tables:

  • Skill checks and difficulty values
  • Combat actions and modifiers
  • Critical injury tables
  • Netrunning procedures

Dice Roller

Built-in dice roller for all game mechanics:

  • Standard dice (d6, d10, etc.)
  • Skill checks with modifiers
  • Damage calculations
  • Critical tables

Notes System

Keep track of campaign information:

  • Create multiple note documents
  • Autosaves to prevent data loss
  • Export and import functionality

NPC Generator

Quickly create NPCs for your game:

  • Random or guided generation
  • Customizable templates
  • Save NPCs for later use

Troubleshooting

Common Issues

  • Layouts not saving: Make sure cookies and local storage are enabled in your browser.
  • Panels not dragging: Try refreshing the page. If the issue persists, clear browser cache.
  • Content not displaying: Check your internet connection and try a different browser.

Data Recovery

The application automatically backs up notes and layouts. To recover data:

  1. Open the Debug Panel from the toolbar menu
  2. Select "Data Recovery" tab
  3. Choose the backup point to restore
