Keyboard Shortcuts

Global Shortcuts

These shortcuts work anywhere in the application and provide quick access to common functions.

Shortcut Action Description
Ctrl + S Save Layout Saves the current panel layout
Ctrl + O Load Layout Opens the layout selection dialog
Ctrl + R Reset Layout Returns to the default panel layout
F1 Show Help Opens the help page
F11 Fullscreen Toggles fullscreen mode
Ctrl + / Show Shortcuts Opens this shortcuts page
Esc Close Dialog Closes the active dialog or modal

Panel Management

These shortcuts help you work with the panel system more efficiently.

Shortcut Action Description
Ctrl + N New Panel Opens the panel selection dialog
Ctrl + W Close Panel Closes the focused panel
Ctrl + M Minimize Panel Minimizes the focused panel
Alt + Tab Next Panel Cycles focus through open panels
Alt + Shift + Tab Previous Panel Cycles focus through open panels in reverse

Feature-Specific Shortcuts

These shortcuts apply to specific panels and features within the application.

Shortcut Feature Action
Ctrl + Enter Dice Roller Roll dice with current settings
Ctrl + D Dice Roller Open dice roller panel
Ctrl + T Tables Open random tables panel
Ctrl + E Notes Open notes editor
Ctrl + B Notes Toggle bold text in notes
Ctrl + I Notes Toggle italic text in notes
Ctrl + 1-9 General Quick access to panels 1-9

Accessibility Shortcuts

These shortcuts are designed to help improve accessibility when using the application.

Shortcut Action Description
Alt + A Accessibility Menu Opens the accessibility options panel
Alt + Z High Contrast Toggles high contrast mode
Alt + + Increase Font Size Makes text larger
Alt + - Decrease Font Size Makes text smaller
Alt + F Focus Mode Simplifies interface to reduce distractions
