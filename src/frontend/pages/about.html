About Cyberpunk RED GM Screen

The Cyberpunk RED GM Screen is a digital tool designed to assist Game Masters running the Cyberpunk RED tabletop role-playing game. This web-based application provides a customizable interface with various panels and tools to enhance your gaming experience.

Key Features

  • Draggable and resizable panels for customization
  • Layout saving and loading system
  • Built-in dice roller with roll history
  • Initiative tracker for combat management
  • Notes panel with rich text editing
  • Rules reference panels for quick lookups
  • NPC generator with customizable options
  • Location generator for quick scene setting
  • Night City map with district information
  • Random encounter generator
  • Theme switching for visual customization

Development

This application is developed as an open-source tool for the Cyberpunk RED community. It's built using modern web technologies including HTML5, CSS3, and JavaScript, with a focus on accessibility and responsive design.

Version Information

Compatibility

The Cyberpunk RED GM Screen is designed to work with modern web browsers including:

  • Chrome 80+
  • Firefox 75+
  • Edge 80+
  • Safari 13.1+

For the best experience, we recommend using the latest version of Chrome or Firefox.
